# 集采联动接口
> last-reviewed: 2026-07-10
> 集采与医保支付标准/进院/定价/基药/DRG/挂网的联动关系
> ※ 本文件只给接口清单，深度分析归 drug-access 对应簇

## 一、定位说明

集采不是孤立的，影响医保/进院/定价/基药/DRG/挂网等多个环节。本文件只给**集采侧结论**和**接口指向**，各环节深度分析归 `drug-access` 对应簇。

**核心原则**：集采侧只给"集采中选价XX元→影响XX"的结论，各环节怎么应对归 access。

## 二、集采 × 医保支付标准（→ drug-access A簇）

### 2.1 联动关系

| 类型 | 医保支付标准确定方式 |
|---|---|
| 国采中选药品 | 以中选价为基准确定支付标准 |
| 集采非中选药品 | 按梯度降价要求调整（≤中选价1.5倍或本省最高中选价） |
| 同通用名未过评药 | 参照中选价/参比制剂按比例确定 |
| 联盟/省级集采 | 按各联盟/省规则确定 |

### 2.2 集采侧结论
- 中选价→医保支付标准基准
- 非中选药须梯度降价→影响支付标准
- 支付标准与挂网价关系→影响患者自付

### 2.3 接口指向
- 医保支付标准深度分析→ `drug-access` A簇（医保准入）
- 集采侧只给："中选价XX元→支付标准XX元→非中选药须降至XX元"

## 三、集采 × 进院（→ drug-access D簇）

### 3.1 联动关系

| 联动点 | 说明 |
|---|---|
| 集采品种进院优先 | 中选品种优先进院，药事会简化/优先 |
| 报量绑定 | 医疗机构报量绑定中选企业→约定采购量 |
| 双通道 | 集采品种双通道药店对接 |
| DRG费用空间 | 集采降价→DRG费用空间变化 |

### 3.2 集采侧结论
- 中标→进院优先+报量绑定→院内份额保障
- 未中标→进院困难→需走院外渠道

### 3.3 接口指向
- 进院策略深度分析→ `drug-access` D簇（医院机构准入）
- 集采侧只给："中选→报量绑定XX量→院内份额XX%"

## 四、集采 × 定价（→ drug-access F簇）

### 4.1 联动关系

| 联动点 | 说明 |
|---|---|
| 集采后价格锚点 | 中标价成为新价格锚点 |
| 全渠道价格联动 | 院内中标价→院外→零售→线上 |
| 价格体系重建 | 集采后需重建分层定价 |
| IRP影响 | 集采价可能影响国际价格参考 |

### 4.2 集采侧结论
- 中标价→新价格锚点→全渠道联动
- 价格体系重建：集采区/非集采区/院外/零售分层定价

### 4.3 接口指向
- 定价策略深度分析→ `drug-access` F簇（定价策略）
- 集采侧只给："中选价XX元→新锚点→全渠道联动至XX元"

## 五、集采 × 基药（→ drug-access B簇）

### 5.1 联动关系

| 联动点 | 说明 |
|---|---|
| 基药品种与集采品种交集 | 部分基药品种同时是集采品种 |
| 基药中选品种供应保障 | 基药身份+集采中选→基层覆盖+供应保障 |
| 集采对基药价格影响 | 集采降价→基药价格变化 |

### 5.2 集采侧结论
- 基药+集采中选→基层覆盖优势+供应保障要求
- 集采降价→基药价格体系变化

### 5.3 接口指向
- 基药准入深度分析→ `drug-access` B簇（基药目录准入）
- 集采侧只给："基药品种XX+集采中选→基层覆盖+供应保障XX"

## 六、集采 × DRG（→ drug-access D簇）

### 6.1 联动关系

| 联动点 | 说明 |
|---|---|
| 集采降价→DRG费用空间 | 集采降价→药品费用下降→DRG费用空间增加 |
| 集采品种在DRG下优势 | 集采品种价格低→DRG下有费用优势 |
| DRG豁免/特病单议 | 部分高值药DRG豁免→集采品种可能不豁免 |

### 6.2 集采侧结论
- 集采降价→DRG费用空间增加→集采品种有费用优势
- DRG下集采品种vs非集采品种的竞争力变化

### 6.3 接口指向
- DRG对进院影响深度分析→ `drug-access` D簇（医院机构准入）
- 集采侧只给："集采降价XX%→DRG费用空间增加XX→集采品种有优势"

## 七、集采 × 挂网（→ drug-access C簇 · 放最后）

### 7.1 联动关系

| 类型 | 挂网规则 |
|---|---|
| 中选药品 | 按中选价/协议价挂网 |
| 未中选药品 | 按梯度降价要求调整→≤中选价1.5倍或本省最高中选价 |
| 集采品种同通用名药 | 以中选价为基准确定医保支付标准 |
| 价格联动 | 集采中选价→全国挂网价联动 |

### 7.2 集采侧结论
- 中选价→直接挂网
- 未中选药→梯度降价挂网（≤中选价1.5倍或本省最高中选价）
- 集采中选价→全国挂网价联动锚点

### 7.3 接口指向
- 挂网准入深度分析→ `drug-access` C簇（集采与招采/挂网）
- 集采侧只给："中选价XX元→挂网XX元→未中选药须降至≤XX元"

## 八、联动接口汇总表

| 联动环节 | 集采侧结论 | 接口指向 |
|---|---|---|
| 医保支付标准 | 中选价→支付标准基准 | drug-access A簇 |
| 进院 | 中选→进院优先+报量绑定 | drug-access D簇 |
| 定价 | 中标价→新锚点→全渠道联动 | drug-access F簇 |
| 基药 | 基药+集采→基层覆盖+供应保障 | drug-access B簇 |
| DRG | 集采降价→DRG费用空间增加 | drug-access D簇 |
| **挂网** | 中选价挂网/未中选梯度降价 | drug-access C簇 |

## 九、边界声明

- 本文件只给**集采侧结论**和**接口指向**
- 各环节深度分析（怎么做医保/怎么进院/怎么定价/怎么挂网）归 `drug-access` 对应簇
- 集采侧不重复 access 的内容，只给"集采影响了什么"的结论
- 挂网准入（首次挂网/窗口期/联动规则/红黄标）归 access C簇，本文件只在集采后挂网调整给结论
