## 一、集采纳入预判模型（★核心创新）

### 1.1 模型目的
回答企业最关心的问题：**我的产品会不会被集采？哪批？什么时候？** 提前1-2年预警，留准备时间。

### 1.2 模型输入

| 输入项 | 说明 | 数据来源 |
|---|---|---|
| 品种类型 | 化药(过评/未过评)/中成药/中药饮片/生物药 | NMPA/药智 |
| 过评企业数(化药) | 通过一致性评价的企业数 | NMPA/药智 |
| 申报企业数 | 同通用名申报/在产企业数 | 药智/米内 |
| 临床用量/市场规模 | 年销售规模/终端销售额 | IQVIA/米内 |
| 竞争格局 | 原研vs仿制/市场份额分布 | IQVIA/米内 |
| 历史纳入规律 | 同类型品种从达标到纳入的典型时间 | 历史批次数据 |

### 1.3 预判逻辑（按品种类型）

#### 1.3.1 化药(过评)纳入预判
```
输入：过评企业数 + 临床用量 + 竞争格局
判断：
  过评≥3家 + 临床用量大 + 竞争充分 → 国采候选(高概率)
  过评≥3家 + 临床用量中 → 关注国采(中概率)
  过评<3家 → 联盟/省级候选
输出：
  纳入概率：高/中/低
  预计层级：国采第N批 / 联盟 / 省级
  预计时间窗口：基于历史批次节奏(约每年1-2批)外推
  预警建议：立即准备投标 / 持续关注 / 暂无近忧
```

#### 1.3.2 中成药纳入预判
```
输入：同通用名企业数 + 临床常用度 + 质量标准
判断：
  多家生产(≥3家) + 临床常用 + 质量标准可评 → 中成药联盟候选
  企业少/临床用量小 → 暂无近忧
输出：
  纳入概率：高/中/低
  预计层级：中成药联盟(湖北) / 省级
  预计时间：关注联盟下一轮品种扩围
```

#### 1.3.3 生物药纳入预判
```
输入：类似药企业数 + 参照药状态 + 临床用量
判断：
  类似药竞争充分(≥3家) + 临床用量大 → 专项/联盟候选
  类似药少 → 暂无近忧
输出：
  纳入概率：高/中/低
  预计层级：专项/联盟/省级
  参考：胰岛素专项标杆
```

### 1.4 历史纳入规律参考
- 化药过评企业数达标→纳入国采的典型间隔：约1-2年（💭 待用历史数据校准）
- 联盟集采品种选择：国采未覆盖+竞争充分+临床常用
- 各批次品种数：25→32→55→45→61→6→60→40→41→62→？（第11批待核实）

## 二、报量策略

### 2.1 报量方式

| 批次/类型 | 报量方式 | 企业应对 |
|---|---|---|
| 第1-10批(化药) | 按通用名报量 | 争取医疗机构报量 |
| **第11批(化药)** | **按厂牌报量**(77%到品牌) | **品牌覆盖度决定报量基础** |
| 中成药联盟 | 按通用名报量 | 医疗机构基于历史使用报量 |
| 胰岛素专项 | 按通用名报量 | 类似药vs参照药分开 |

### 2.2 报量评估要素
1. **首年约定采购量**：医疗机构报量是否合理？与历史用量对比
2. **产能匹配**：报量vs年产能vs产能利用率→能否满足供应
3. **配送匹配**：报量分布vs配送网络覆盖
4. **原料药供应**：报量vs原料药供应能力

### 2.3 按厂牌报量应对（第11批）
- 品牌覆盖度直接影响报量基础
- 提前做医疗机构品牌覆盖工作
- 新品牌/新上市产品报量可能不足→评估是否值得投标

## 三、报价策略

### 3.1 成本底线测算（报价下限）

```
成本底线 = 原料药成本 + 制造成本 + 配送成本 + 质量成本 + 合理利润
                    ↓
          低于此价不投（亏损风险）
```

| 成本项 | 测算要点 |
|---|---|
| 原料药成本 | 原料药采购价+波动风险+替代来源 |
| 制造成本 | 生产线成本/批+产能利用率 |
| 配送成本 | 全国/区域配送网络+冷链(生物药) |
| 质量成本 | 一致性评价/质量标准/道地药材(中成药) |
| 合理利润 | 企业可接受的最低利润率 |

### 3.2 降幅测算

| 降幅基准 | 说明 |
|---||
| vs当前挂网价 | 最常用基准 |
| vs最高有效申报价 | 部分批次使用 |
| vs参比制剂价格 | 化药过评药 |
| vs参照药价格 | 生物药 |

- 历史同类品种降幅参考：第1批平均52%→后续批次约48-60%（具体品种须查证）
- 品种类型差异：化药降幅大→中成药降幅相对小（质量评分加成）

### 3.3 中标概率评估

| 评估因素 | 说明 |
|---|---|
| 竞争对手数 | >10家且成本无优势→中标概率低 |
| 报价vs历史中标价区间 | 在历史区间内→概率中；低于区间→概率高但亏损风险 |
| 锚点规则(第11批) | 报价在锚点附近→概率高；偏离锚点→概率低 |
| 产能/供应保障 | 供应能力不足→即使中标也可能断供 |

### 3.4 锚点规则应对（第11批）
- 锚点：综合报价情况确定锚点价格
- 价差在允许范围内中选
- **报价策略**：从"最低价"转向"锚点附近"
- 需预判竞争对手报价→确定锚点区间→报价落在区间内

## 四、中选规则解读

### 4.1 各层集采中选规则

| 集采层级 | 中选规则 | 说明 |
|---|---|---|
| 国采(第11批) | 锚点规则 | 价差计算锚点，非简单最低价 |
| 国采(早期) | 最低价/降幅/综合 | 演进至锚点 |
| 中成药联盟 | 降幅+质量综合评分 | 30%量给降幅最高 |
| 跨省联盟 | 各联盟不同 | 降幅/绝对价/综合/联动 |
| 省级集采 | 各省不同 | 规则差异大 |

### 4.2 中选区分配

| 中选区 | 说明 |
|---|---|
| 主供区 | 中选企业主要供应区域 |
| 非主供区 | 中选企业非主要供应区域（≤中选价1.5倍或本省最高中选价） |
| 备供区 | 备用供应区域 |

## 五、资质门槛自评（第11批新增3条）

### 5.1 三条新增资质
1. **2年以上同类型制剂生产经验**：申报企业/受委托生产企业
2. **GMP符合性检查通过的生产线**：申报药品须在通过检查的生产线生产
3. **2年内不违反GMP要求**：生产线2年内无GMP违规

### 5.2 资质自评清单
- [ ] 同类型制剂生产经验≥2年？
- [ ] 生产线条线GMP符合性检查通过？
- [ ] 生产线2年内无GMP违规记录？
- [ ] 一致性评价状态(化药)？
- [ ] 质量标准(中成药)？
- [ ] 参照药状态(生物药)？

> 任一项不满足→不投（资质整改优先）

## 六、投标决策树（★核心创新）

```
Step1 资质门槛自评
  ├─ 不满足 → 不投（资质整改）
  └─ 满足 → Step2

Step2 成本底线测算
  ├─ 报价 < 成本底线 → 不投（亏损风险）
  └─ 报价 ≥ 成本底线 → Step3

Step3 竞争格局评估
  ├─ 竞争对手>10家 且 成本无优势 → 谨慎投（中标概率低）
  ├─ 竞争对手适中 或 成本有优势 → Step4
  └─ 竞争对手少 → Step4

Step4 供应保障评估
  ├─ 产能/配送不满足报量 → 不投（断供风险）
  └─ 产能/配送满足 → Step5

Step5 战略考量（量价权衡）
  ├─ 战略品种 → 投（保份额，可接受低利润）
  ├─ 非战略品种 + 保价格更重要 → 不投（保价格，走院外）
  └─ 非战略品种 + 保份额更重要 → 投

→ 输出：
  投/不投
  报价区间：[成本底线, 锚点附近/历史中标价区间]
  报量策略：[争取报量/按实际产能报量]
  风险提示：[亏损风险/断供风险/未中标风险]
```

### 6.1 决策树使用说明
- 5步顺序执行，任一步"不投"则终止
- 每步判断需基于事实数据，不凭感觉
- 最终输出"投/不投+报价区间+报量策略"
- 决策树是框架，具体判断需结合企业战略和实际情况

## 七、按品种类型差异化投标要点

| 品种类型 | 投标关键差异 |
|---|---|
| 化药(过评) | 锚点规则(第11批)+按厂牌报量+资质3条 |
| 化药(未过评) | 各省分别研究+价格参照过评药60% |
| 中成药 | 降幅+质量综合评分+30%量给降幅最高+品牌影响70%自主选择 |
| 中药饮片 | 规格标准化+道地药材+储存运输 |
| 生物药 | 参照药选择+类似药竞争+冷链供应 |
