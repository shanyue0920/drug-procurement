## 一、中选后执行流程

### 1.1 执行流程总览

```
中选公告 → 中选价挂网 → 医疗机构报量绑定 → 合同签约 → 执行供应 → 协议期满 → 接续
```

### 1.2 各环节要点

| 环节 | 要点 |
|---|---|
| 中选公告 | 联采办/联盟发布中选结果公告 |
| 中选价挂网 | 中选价直接挂网，各省招采平台更新 |
| 报量绑定 | 医疗机构报量绑定中选企业→约定采购量 |
| 合同签约 | 医疗机构与中选企业签订购销合同 |
| 执行供应 | 协议期内按约定采购量供应 |
| 协议期满 | 到期后进入接续采购(P8) |

### 1.3 协议期
- 通常1-3年（各批次/各品种不同）
- 协议期内价格锁定（中选价）
- 协议期满后接续采购

### 1.4 "三进"扩围（第11批）
- **进基层**：基层医疗机构参与报量采购→渠道下沉
- **进民营**：民营医疗机构参加报量→渠道扩展
- **进零售药店**：零售药店参加报量→院外渠道
- **企业影响**：供应范围扩大→需布局多渠道供应保障

## 二、合同履约管理

### 2.1 履约率监控
- **监控指标**：实际采购量 vs 约定采购量
- **履约率计算**：实际采购量 / 约定采购量 × 100%
- **预警线**：履约率 < 80% 预警（具体各省标准不同）

### 2.2 履约率现状
- 2023年通报：合同履约率约82%
- 短缺药品中约60%为集采品种
- 低于成本中标→断供→短缺的链条

### 2.3 履约不达标后果

| 后果 | 说明 |
|---|---|
| 信用扣分 | 医药价格和招采信用评价扣分 |
| 暂停挂网 | 严重不达标暂停该企业挂网 |
| 取消投标资格 | 屡次不达标取消后续集采投标资格 |
| 公开通报 | 医保局公开通报失信企业 |
| 信用黑名单 | 严重失信进入黑名单 |

### 2.4 履约保障策略
1. **产能规划**：年产能 > 约定采购量（留安全余量）
2. **配送网络**：覆盖所有报量省份/机构
3. **库存管理**：安全库存+应急预案
4. **原料药供应**：多源采购+长期合同锁定
5. **监控系统**：实时监控各省履约率→预警→及时补货

## 三、供应保障能力评估

### 3.1 产能评估

| 评估项 | 要点 |
|---|---|
| 年产能 | 年产能 vs 约定采购量→产能利用率 |
| 产能利用率 | 当前利用率→中标后能否满足 |
| 扩产计划 | 如需扩产→时间/投资/可行性 |
| 生产线状态 | GMP符合性/生产线数量/批次产能 |

### 3.2 配送评估

| 评估项 | 要点 |
|---|---|
| 配送网络 | 覆盖所有报量省份/机构 |
| 冷链能力 | 生物药/部分化药需冷链 |
| 偏远地区 | 偏远地区配送成本/时效 |
| 配送商 | 配送商资质/覆盖/响应速度 |

### 3.3 库存评估

| 评估项 | 要点 |
|---|---|
| 安全库存 | 覆盖突发需求的库存量 |
| 库存周转 | 库存周转率/保质期管理 |
| 断供风险点 | 原料药断供/生产线故障/配送中断 |

### 3.4 原料药供应评估

| 评估项 | 要点 |
|---|---|
| 原料药来源 | 来源稳定性/供应商集中度 |
| 原料药涨价 | 涨价风险/长期合同锁定 |
| 替代来源 | 替代供应商/替代原料药 |
| 原料药质量 | 原料药质量一致性 |

## 四、断供风险与应对

### 4.1 断供常见原因

| 原因 | 说明 |
|---|---|
| 低于成本中标 | 报价<成本→亏损→停产→断供 |
| 原料药涨价 | 原料药成本上升→亏损→断供 |
| 产能不足 | 产能<约定采购量→供应不上 |
| 配送不到位 | 配送网络覆盖不足→部分机构断供 |
| 生产线故障 | 生产线故障/停产检修→断供 |
| 质量问题 | 质量问题→召回→断供 |

### 4.2 断供后果

```
断供 → 信用黑名单 → 暂停挂网 → 取消投标资格 → 公开通报
         ↓
    严重影响企业后续所有集采参与
```

- **信用黑名单**：医药价格和招采信用评价失信
- **暂停挂网**：该企业产品暂停挂网→影响非集采品种
- **取消投标资格**：取消后续集采投标资格
- **公开通报**：医保局公开通报→声誉损失

### 4.3 断供应对

| 阶段 | 应对 |
|---|---|
| 预警 | 实时监控履约率/库存/原料药→提前预警 |
| 主动沟通 | 发现供应困难→主动与医保局/医疗机构沟通 |
| 申请调整 | 申请调整供应量/配送区域/协议期 |
| 应急预案 | 紧急补货/替代供应/临时调配 |
| 记录留痕 | 供应困难原因/沟通记录/应对措施留痕 |

### 4.4 断供预防（投标前评估）
- 投标决策树Step4：产能/配送不满足→不投（断供风险）
- 成本底线测算：报价≥成本底线→避免低于成本中标
- 原料药供应评估：多源采购+长期合同→避免原料药断供

## 五、按品种类型执行差异

| 品种类型 | 执行关键差异 |
|---|---|
| 化药 | 标准化生产→批次差异小→供应保障相对容易 |
| 中成药 | 道地药材供应·批次差异·质量一致性→供应保障难度中 |
| 中药饮片 | 规格标准化·储存运输(防潮/防霉/防虫)→供应保障难度高 |
| 生物药 | 冷链配送·参照药切换·免疫原性→供应保障难度高 |

### 5.1 中成药执行特殊性
- 道地药材供应：不同批次原料药质量差异→需控制原料来源
- 批次差异：同一企业不同批次产品质量差异→需批次管理
- 质量一致性：质量标准不统一→需企业内控标准

### 5.2 中药饮片执行特殊性
- 储存运输：防潮/防霉/防虫→仓储条件要求高
- 规格标准化：不同产地/炮制/等级→规格一致性难
- 配送：饮片体积大/重量大→配送成本高

### 5.3 生物药执行特殊性
- 冷链配送：全程冷链→配送成本高/断链风险
- 参照药切换：患者从参照药切换类似药→需医生指导
- 免疫原性：类似药vs参照药免疫原性差异→需监测
