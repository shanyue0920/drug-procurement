## 案例一 · 胰岛素专项集采（第六批国采 · 2021 · 生物药）

### 1. 元数据头
- 分析对象：胰岛素（二代+三代，16 个通用名品种）
- 品种类型：生物药（胰岛素专项）
- 分析视角：集采后竞争格局与应对（P5）+ 投标策略（P3）
- 数据截止：2026-07；证据等级：L1（联采办/国家医保局通报、权威媒体）

### 2. 执行摘要
1. 🔍 第六批国采（胰岛素专项）2021-11-26 开标、11-30 公布中选，涵盖二代/三代胰岛素 6 个采购组、81 个产品，11 家企业全部中选（来源：国家医保局/上海阳光医药采购网，2021-11-30；访问 2026-07）。
2. 🔍 平均降价 48%，最高降幅 74%（礼来精蛋白锌重组赖脯胰岛素 25R，报 18.89 元）；首年采购需求约 2.1 亿支、涉及金额约 170 亿元，预计年节约 90 亿元（来源：国家医保局通报、21 世纪经济报道 2021-11-26）。
3. 🔍 甘李药业 6 产品全中、平均降幅约 65%，3 款为同组最低价；通化东宝 5 产品中标、平均降幅 41%（来源：科创板日报/人民号 2021-11-26）。
4. 💭 生物药特殊性：参照药选择与类似药替代节奏慢于化药，后格局以「价格分层+品牌」为主。

### 3. 主体（按生命周期 + P3/P5）
- **纵向对比（本次 vs 上次）**：胰岛素为首个生物药专项国采，此前国采以化药为主；规则按「采购组+A/B/C 类量价挂钩」设计，无化药式一致性评价门槛。
- **投标策略（P3）**：甘李采取「以价换量」——报全组最低价争 A 类（低价多量）；通化东宝取 B 类稳健策略。
- **集采后格局（P5）**：中选企业扩产保供应；原研（诺和诺德/礼来/赛诺菲）全中，凭品牌+循证守份额；未中标渠道转移不适用（本次全员中标）。

### 4. 关键结论与行动建议
- 对药企：生物药集采「降幅可控但量大」，产能与冷链供应保障是关键；甘李式激进报价适用于成本领先者。
- 行动：① 立即核对自身在 A/B/C 类的定位；② 短期做产能与配送预案；③ 中期布局差异化（剂型/规格/患者服务）。

### 5. 不确定性 · 待核实清单
- ⚠️ 待核实：各企业具体中选价按品规的明细清单，须以联采办《中选结果通知》原文逐条复核（本示范取已披露代表品规）。
- 💭 推测：二代/三代替代节奏须以最新 IQVIA/米内份额数据更新，旧内存不适用。

### 6. 信息来源与证据等级
- 国家医保局/上海阳光医药采购网《全国药品集中采购（胰岛素专项）中选结果通知》（2021-11-30）🔍 P0 L1
- 21 世纪经济报道《300 亿市场变局：胰岛素国家集采最高降 74%》（2021-11-26，URL：21jingji.com）🔍 L1
- 科创板日报/人民号《胰岛素国采结果出炉：甘李或成最大赢家》（2021-11-26）🔍 L2

## 案例二 · 硫酸氢氯吡格雷（4+7 扩围 · 2019 · 化药过评·原研 vs 仿制）

### 1. 元数据头
- 分析对象：硫酸氢氯吡格雷片（75mg 主品规）
- 品种类型：化药（已过评/原研+首仿）
- 分析视角：投标策略（P3）+ 集采后格局（P5·原研应对/未中标渠道转移）
- 数据截止：2026-07；证据等级：L1（联盟采购公告、权威媒体）

### 2. 执行摘要
1. 🔍 2019-09-24「4+7」全国扩围（25 省+兵团）开标，氯吡格雷 75mg 主品规中标企业：石药欧意 2.44 元/片、赛诺菲（原研）2.55 元/片、乐普 2.98 元/片（来源：赛柏蓝/健康界 2019-09-25）。
2. 🔍 信立泰（首仿泰嘉）报价 3.13 元/片出局——其首轮 4+7 中标价 3.18 元/片，本轮降价至 3.13 仍高于对手（来源：每日经济新闻/人民号 2019-09-25）。
3. 🔍 原研赛诺菲以「仿制药价格」应战中标，因其首轮未中标致销量下滑近 9 成（来源：界面新闻，转引健康界）。
4. 💭 市场：氯吡格雷为百亿级抗血栓大品种，原研波立维曾占市场首位。

### 3. 主体（按 P3/P5）
- **投标决策树（P3e）**：信立泰败因=报价高于锚点且成本/策略误判；赛诺菲触发「原研防御性降价」避免出局。
- **集采后格局（P5）**：信立泰出局→启动渠道转移（院外/DTP/零售）+ 加速创新转型（公告 2.5-5 亿回购）；赛诺菲以原研品牌+集采中标守量。
- **横向对比**：本品种走「省级联盟/扩围」非国采主线（当时过评企业数有限），规则为「最低价+多家中标瓜分 70% 量」。

### 4. 关键结论与行动建议
- 对药企：原研在存量大、集采未中即销量崩塌时，会「跳水」保份额；仿制药企不可低估原研防御报价。
- 行动：① 投标前做对手报价情景模拟（含原研）；② 未中标立即启动院外/零售预案；③ 成本底线须优于潜在对手最低报价。

### 5. 不确定性 · 待核实清单
- ⚠️ 待核实：本轮各企业约定采购量分配比例（A/B 类量）须以联盟采购文件原文复核。
- 💭 推测：信立泰出局后院外渠道实际放量数据须以企业年报/米内更新，非本示范范围。

### 6. 信息来源与证据等级
- 赛柏蓝/健康界《47 全国扩围 中标结果流出》（2019-09-25）🔍 L1/L2
- 每日经济新闻/人民号《带量采购"冲击波"：信立泰流标》（2019-09-25）🔍 L2
- 界面新闻/健康界（赛诺菲降价动因）🔍 L2

## 案例三 · 第十一批国家集采（2025 · 锚点规则·政策追踪/投标策略）

### 1. 元数据头
- 分析对象：第十一批国采（55 个品种、162 个品规）
- 品种类型：化药为主（口服常释/注射剂）
- 分析视角：政策动态追踪（P7）+ 投标策略（P3·锚点规则）
- 数据截止：2026-07；证据等级：L1（联采办文件、国家医保局）

### 2. 执行摘要
1. 🔍 第 11 批集采文件 2025-09-20 发布、2025-10-21 上海开标、10-28 公示拟中选；覆盖 55 品种（来源：联采办《全国药品集中采购文件》2025-09-20；国家医保局）。
2. 🔍 锚点规则（核心变化）：以「入围企业单位可比价均值 50%」与「最低价」取高值为锚点；报价超过锚点 1.8 倍须降至 1.8 倍内方可中选（来源：国家医保局/中青报 2025-09-21、界面新闻 2025-09-20）。
3. 🔍 规模：4.6 万家机构报量、445 家企业 794 产品投标、272 家企业 453 产品拟中选，中选率约 60%；第一轮平均降幅约 75%，超七成品种降幅 70% 以上（来源：医药魔方/易联招采 2025-10-31）。
4. 🔍 资质新增 3 条（2 年以上同类型制剂生产经验/GMP 符合性检查/2 年不违反 GMP）；按厂牌报量（约 77% 报至品牌）。

### 3. 主体（按 P3/P7）
- **纵向对比（第 10 批 → 第 11 批）**：锚点由「最低价」改为「入围均价 50% 与最低价取高值」，熔断上限抬高、防异常低价（修改类）；新增「低价声明/不低于成本承诺」（新增类）；规则导向由「唯低价」转向「防内卷+保质量」（制度化）。
- **投标决策树（P3e）**：锚点抬高→激进最低价策略边际收益下降，成本底线与质控证明成为新约束；需先核验 3 条资质门槛。

### 4. 关键结论与行动建议
- 对药企：第 11 批「反内卷」下，不必非报最低价；重点在成本证明+质控+按厂牌报量争取。
- 行动：① 立即核对 3 条资质门槛；② 报价模拟以新锚点为上限；③ 中标后按厂牌绑定报量做供应保障。

### 5. 不确定性 · 待核实清单
- ⚠️ 待核实：55 品种逐品种中选价与降幅须以联采办拟中选结果附件逐条复核（本示范取汇总量）。
- 💭 推测：锚点对个别品种的实际影响（约 1/4 触发调高）须以官方模拟数据更新。

### 6. 信息来源与证据等级
- 联采办《全国药品集中采购文件（第十一批）》（2025-09-20）🔍 P0 L1
- 中青报《集采新规力保"降价不降质"》（2025-09-21）🔍 L1
- 界面新闻《新一批国家药品集采将如何"防内卷、保质量"？》（2025-09-20）🔍 L2
- 医药魔方/易联招采《第十一批国采拟中选分析》（2025-10-31）🔍 L2
