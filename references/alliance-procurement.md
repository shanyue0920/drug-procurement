> 4层集采体系的第二层(全国联盟)·第三层(跨省联盟)·第四层(省级)

## 一、第二层：全国联盟采购

### 1.1 概述
- 国家医保局指导，地方牵头组织
- 2025年预计约20个全国联盟采购
- 聚焦专业特色品种：中成药/中药饮片/生物药/高值耗材
- 与国采互补：国采覆盖化药过评品种，联盟覆盖国采未覆盖品种

### 1.2 中成药联盟集采（重点）

#### 湖北联盟（中成药集采主力）
- **牵头省**：湖北
- **覆盖品种**：中成药（口服/注射）
- **规则特色**：
  - 无法做一致性评价→以质量标准+临床认可度为替代门槛
  - "30%量分给降幅最高的中选企业，70%由医疗机构自主选择"
  - 综合评分：降幅+质量+企业信用+供应保障
- **难点**：
  - 各家产品质量标准不统一
  - 中药材受道地产区影响，价格和质量差异大
  - 同一通用名下不同企业产品差异大

#### 中成药集采特殊性（vs化药）
| 维度 | 化药集采 | 中成药集采 |
|---|---|---|
| 一致性评价 | 有（门槛） | 无（以质量标准替代） |
| 质量标准 | 统一 | 不统一（各家不同） |
| 中选规则 | 锚点/最低价 | 降幅+质量综合评分 |
| 量分配 | 报量绑定 | 30%降幅最高+70%自主选择 |
| 价格差异 | 同通用名价格趋同 | 同通用名价格差异大（道地药材/工艺） |

### 1.3 中药饮片联盟集采（2025年新开展）
- **新开展**：2025年启动
- **聚焦品种**：临床常用饮片品种
- **难点**：
  - 规格标准化（炮制方法/产地/等级差异大）
  - 道地药材影响（同品种不同产地质量价差大）
  - 储存运输（防潮/防霉/防虫）
- **规则探索中**：各省/各联盟规则可能不同

### 1.4 生物药联盟集采
- **江西干扰素联盟**：干扰素集采为标志
- **其他生物药**：多为省级探索性采购
- **参照药选择关键**：生物类似药vs参照药的价格对标
- **生物类似药替代**：集采加速类似药对参照药替代

### 1.5 全国联盟动态追踪
- 联盟数量：2025年约20个
- 联盟品种：中成药/中药饮片/生物药/特色品种
- 动态变化：新联盟组建/品种扩围/规则调整
- ⚠️ 使用前须联网核实最新联盟清单

### 2.2 联盟规则差异（核心）

#### 2.2.1 品种选择差异
- 各联盟品种选择不同
- 有的接续国采到期品种，有的做国采未覆盖品种
- 有的聚焦特色品种（中成药/注射剂/生物药）

#### 2.2.2 中选规则差异
| 规则类型 | 说明 | 典型联盟 |
|---|---|---|
| 降幅达标 | 降幅达到一定比例中选 | 部分中成药联盟 |
| 绝对价格 | 绝对价格最低中选 | 部分化药联盟 |
| 综合评分 | 降幅+质量+信用+供应综合评分 | 中成药联盟 |
| 带量联动 | 参照其他集采中选价联动 | 四川联盟(带量联动) |
| 锚点规则 | 价差锚点范围内中选 | 部分新批次联盟 |

#### 2.2.3 量分配规则差异
| 量分配方式 | 说明 | 典型 |
|---|---|---|
| 报量绑定 | 按医疗机构报量绑定中选企业 | 多数联盟 |
| 降幅最高优先 | 降幅最高企业获得更多量 | 中成药联盟(30%) |
| 自主选择 | 医疗机构在中选企业中自主选择 | 中成药联盟(70%) |
| 按比例分配 | 按中选顺位/报价分配量 | 部分联盟 |

### 2.3 药企跨省联盟应对要点
- **按联盟分别研究规则**：不能一套策略打天下
- **覆盖省份差异**：同一品种在不同联盟的竞争格局不同
- **接续vs新采**：接续品种有历史价格基准，新采品种需重新定价
- **联动影响**：一个联盟中标→可能影响其他联盟/省份价格联动

## 三、第四层：省级集采

### 3.1 概述
- 各省自行组织
- 聚焦前三层未覆盖品种
- 各省规则差异更大

### 3.2 省级集采追踪要点
- 各省集采品种清单
- 各省集采时间安排
- 各省集采规则（中选标准/量分配/执行周期）
- 各省执行落地差异

### 3.3 省级集采与前三层关系
| 关系 | 说明 |
|---|---|
| 补充 | 覆盖国采/联盟未覆盖品种 |
| 接续 | 国采/联盟到期品种省级接续 |
| 联动 | 参照国采/联盟中选价联动 |
| 探索 | 省级先行探索→成熟后上升为联盟/国采 |

## 四、按品种类型的联盟/省级集采路径

| 品种类型 | 联盟/省级集采主战场 | 关键差异 |
|---|---|---|
| **化药(过评)** | 国采为主→联盟接续→省级补充 | 国采覆盖后联盟做接续 |
| **化药(未过评)** | 联盟/省级为主 | 无国采门槛；各省规则不同 |
| **中成药** | 中成药联盟(湖北)为主→省级补充 | 无一致性评价；降幅+质量评分 |
| **中药饮片** | 中药饮片联盟(新开展) | 规则探索中；道地药材+规格标准化 |
| **生物药** | 胰岛素专项(国采)+干扰素联盟+省级探索 | 参照药选择关键；类似药替代 |

## 五、联盟集采投标策略要点

### 5.1 联盟识别
1. 你的品种在哪个联盟覆盖范围内？
2. 该联盟牵头省是谁？参与省有哪些？
3. 该联盟规则是什么？（降幅/绝对价/综合评分/联动）

### 5.2 报量策略
- 联盟报量vs国采报量差异
- 跨省报量→跨省供应保障
- 按厂牌报量vs按通用名报量

### 5.3 报价策略
- 参照该联盟历史中选价
- 参照国采同品种中选价（联动参考）
- 联盟间价格联动风险评估

### 5.4 中选后执行
- 跨省供应保障（配送网络/库存）
- 各省落地节奏差异
- 履约率各省分别管理
