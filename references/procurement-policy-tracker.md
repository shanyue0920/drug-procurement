### 2.2 规则演进主线

#### 2.2.1 品种遴选演进
| 阶段 | 批次 | 特点 |
|---|---|---|
| 试点期 | 第1-3批 | 过评品种为主，竞争充分的化药 |
| 规模化 | 第4-7批 | 扩大品种范围，注射剂占比提升，胰岛素专项 |
| 优化期 | 第8-10批 | 规则优化，中选规则更精细化 |
| 制度化 | 第11批 | 品种遴选规则明确化，资质门槛制度化 |

#### 2.2.2 报量方式演进
- 第1-10批：按通用名报量
- 第11批：按厂牌报量（77%报到品牌）

#### 2.2.3 中选规则演进
- 第1-4批：最低价中标为主
- 第5-7批：最低价+降幅达标+综合评分
- 第8-10批：多家中选+价差控制
- 第11批：锚点规则

#### 2.2.4 资质门槛演进
- 第1-10批：基本资质（过评/生产许可）
- 第11批：新增3条（2年经验+GMP符合性+2年无违规）

#### 2.2.5 执行范围演进
- 第1批：4+7城市公立医院
- 第1批扩围：全国公立医院
- 第2-10批：全国公立医院+部分民营自愿
- 第11批：三进（基层+民营+零售药店）

## 三·补 纵向对比框架（每次必做）

### 对比逻辑
- **纵向 = 时间轴对比**：本次集采 vs 上一次集采（上一批/上一轮/上一版）
- **每次必做**：不是"涉及变化时才做"，而是每次讨论集采都做
- **即使无变化也要输出**：标注"本次与上次一致，无变化"，说明延续性/可预期性含义

### 纵向对比表模板
```
## 集采纵向对比（{本次} vs {上次}）

| 对比维度 | 上次（批次/年份） | 本次（批次/年份） | 变化类型 | 对企业影响 |
|---|---|---|---|---|
| 品种遴选 | ... | ... | 新增/删除/修改/制度化 | ... |
| 报量方式 | ... | ... | ... | ... |
| 中选规则 | ... | ... | ... | ... |
| 资质门槛 | ... | ... | ... | ... |
| 执行范围 | ... | ... | ... | ... |
| 量分配 | ... | ... | ... | ... |
| 接续机制 | ... | ... | ... | ... |
```

### 纵向对比结论（必须输出）
- 规则变了什么（变化类型+变化内容）
- 对企业影响（直接影响+间接影响）
- 企业要做什么（立即/短期/中期行动）

## 三·补2 横向对比框架（每次必做）

### 对比逻辑
- **横向 = 空间轴对比**：同一品种/同类品种在不同层级、不同联盟、不同省之间
- **每次必做**：不是"涉及地域差异时才做"，而是每次讨论集采都做
- **即使只涉及一个层级也要做假设对比**：与"如果走其他层级/联盟/省会怎样"对比

### 横向对比的三个类型

| 横向对比类型 | 说明 | 示例 |
|---|---|---|
| **跨层级对比** | 国采 vs 联盟 vs 省级 | 某化药：国采锚点规则 vs 联盟降幅规则 vs 省级绝对价规则 |
| **跨联盟对比** | 不同联盟之间 | 某中成药：湖北联盟 vs 京津冀联盟 vs 广东联盟 |
| **跨省对比** | 不同省之间 | 某品种：四川 vs 广东 vs 浙江的执行节奏/履约要求 |

## 五、前瞻预判

### 5.1 下一批国采可能纳入什么品种？
**预判依据**：
- 过评企业数≥3家的品种（化药）
- 临床用量大、竞争充分的品种
- 国采未覆盖但联盟已采过的品种（上升为国采）

**预判方法**：
1. 查NMPA/药智：过评企业数≥3家的品种清单
2. 查临床用量：终端销售额排名靠前的品种
3. 查历史：联盟集采已覆盖但国采未覆盖的品种
4. 交叉比对→预判下一批候选品种

### 5.2 规则可能怎么变？
**演进趋势外推**：
- 品种遴选：越来越透明化/制度化
- 报量：从通用名→厂牌→更精细化
- 中选：从最低价→锚点→更复杂的多维评价
- 资质：门槛越来越高（生产经验/GMP/合规）
- 执行：范围越来越广（三进→更多渠道）

### 5.3 联盟集采下一阶段重点
- 中成药联盟：品种扩围（更多中成药纳入）
- 中药饮片联盟：规则完善（从探索到制度化）
- 生物药联盟：品种扩围（从胰岛素/干扰素→更多生物药）
- 特色联盟：高值耗材/特色品种

### 9.2 生命周期指导意见要点
- **全路径推演**：不只看当前阶段，推演到接续/续约
- **时间节点**：给具体预计时间（基于预判模型+批次节奏），不只说"未来"
- **各阶段策略**：每个阶段给可操作的建议，不给空话
- **长期演变**：价格/份额/渠道三个维度的长期趋势预判
